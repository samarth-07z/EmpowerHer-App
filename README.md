#### 1. **Clone the Repository**
```bash

cd empowerher
```

#### 2. **Install Dependencies**
```bash

# or
yarn install
```

#### 3. **Set up Google Maps API** (Optional)

- Create a new project or select existing one

#### 4. **Start Development Server**
```bash

# or
yarn dev
```

#### 5. **Open Your Browser**
